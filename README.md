# Employee Salary Analysis Dashboard

## Project Overview

This project analyzes employee salary data and presents insights through a dashboard-style data analysis workflow. It is designed to demonstrate data cleaning, exploratory data analysis, visualization, and business reporting skills using Python.

---

## Business Problem

Organizations need to understand compensation patterns across roles, departments, experience levels, and employee groups. This project shows how salary data can be reviewed to identify trends, compare averages, and support better decision-making.
